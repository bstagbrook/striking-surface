## Verification Standard (formal, not peer review)

Every pair `(A . B)` must be grounded by one of:
1. **Axiom** — accepted primitive (cite 000-primitives)
2. **Citation** — verified in another strike table (cite file + pair)
3. **Computation** — mechanically reproduced (cite tool + method)
4. **Measurement** — published empirical data (cite paper, author, year)

Every pair gets a `;;; VERIFIED:` comment. No exceptions.
The citation network across strike tables IS the formal verification.
